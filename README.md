This project is an interactive quiz web app that allows users to test their knowledge through multiple-choice questions. It is built using HTML, CSS, JavaScript, JSON, and Bootstrap, with a responsive and user-friendly design.
FEATURES:-
Multiple-choice questions (MCQs)
Real-time score calculation
Responsive design (mobile-friendly)
Dynamic question loading using JSON
Restart quiz option
TECHNOLOGIES USED :-
HTML
CSS
Bootstrap
JavaScript
JSON
